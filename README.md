### Project One - 10 Businesses Financial Overview

##### Project One Objective
In this project, the objective is to create a single dashboard for 10 different businesses owned by an investor. This investor is interested in knowing which of the business in his porfolio performing optimally, track the daily transactions from each business, and identify at a glance total profit/loss from all the business.

##### Project One Tools
In other for an assessment and solution to the investor's inquiries, the following tools were used;
* Google Sheet
* Microsoft Power BI
* Microsoft Power Query
* M Langauage
* DAX

##### Project One Workflow
The project takes shape from Google Sheet, as google sheet serves as a spreedsheet that records all the daily transactions remotely. Afterwards, the information from google sheet is imported and transformed on powerbi using power query and m language, which now makes the data ready for restructuring with DAX and then graphical and illustrative visualisation and explaination.

### Project Four - Whatsapp Business Messenger API Automation

##### Project Four Objective
In this project, the objective is to propagate the campaign of a politician on whatsapp and other social media channels. With a focus on just whatsapp, the objective was to create election awareness for a particular candidate to voters in anambra north and measure voters reaction through a automated feedback system.

##### Project Four Tools
In other for an assessment and solution to the investor's inquiries, the following tools were used;
* Microsoft Excel
* Google Sheet
* Whatsapp Business API
* Automation Web Application - Make
* Openshot Video Editor
* Canva

##### Project Four Workflow
The project start with an archive of voter's information for the location in consideration. The data went through a refinery process on power query before been exported to google sheet for remote accessibility and feedback tracking. With the data on google sheet, a whatsapp business api is requested. The api key from the request and the google sheet id of voters contact was deployed into an automated machine adjusted to send canva's graphics and edited short video clips. Thus, tracking voter's interactions.
